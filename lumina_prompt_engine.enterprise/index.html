Lumina Prompt Engine

Multi-model prompt orchestration with a Next.js dashboard, FastAPI backend, and optional CrewAI AMP deployment.

Define a Persona, generate six model-specific prompts in parallel, score each with a Judge-LLM, and export ranked results — locally or in production on Vercel + CrewAI.

  • Backend in backend/ — router, DeepEval judge, shared pipeline
  • Frontend in frontend/ — Persona Studio, matrix, PDF/Excel exports
  • CrewAI in crewai/ — same pipeline for AMP kickoff API
