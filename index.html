About

I'm Navendu Brajesh, an AI/ML engineer focused on context engineering: the discipline of curating everything an LLM sees so agents stay reliable as sessions grow. My open-source work distills production patterns — progressive disclosure, observation masking, compaction, harness design — into reusable Agent Skills that work across tools like GitHub Copilot, Cursor, Kiro, Antigravity, and Amazon Q. I write about these ideas on Medium and build them in the open on GitHub.
